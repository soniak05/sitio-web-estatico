# sitio-web-estatico

## Presentación del proyecto
Este repositorio contiene un **sitio web estático** desarrollado para la práctica del RA3: *Publicación de sitios estáticos y documentación online*.  

El objetivo ha sido:
1. Desarrollar un sitio web básico utilizando **HTML y CSS**.  
2. Publicar el sitio automáticamente mediante **GitHub Pages**.

## Qué se ha hecho
- Creación de un archivo **`index.html`** como página principal.  
- Uso de **etiquetas semánticas** para organizar el contenido:  
  `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`.  
- Personalización visual con **CSS**, incluyendo:  
  - Tipografías externas con **Google Fonts**.  
  - Colores, gradientes y efectos hover.  
  - Bloques destacados y tarjetas para organizar la información.  
- Despliegue del sitio mediante **GitHub Pages**.
